快速开始
========

1. 安装依赖:
   cd colorization_project
   pip install -r requirements.txt

2. 下载数据:
   python -m data.download_data --dataset coco

3. 训练模型:
   python train.py --dataset coco --batch_size 32 --use_amp

4. 测试模型:
   python test.py --checkpoint ../checkpoints/best_model.pth

5. 推理上色:
   python examples.py
